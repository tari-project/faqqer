# AI Help Bot (New Generation)

This directory will house the new, self-learning, multi-channel AI help system that is replacing the legacy `Faqqer` bot.

Refer to the root-level [Design Document (docs/DESIGN.md)](../docs/DESIGN.md) for the project's framework selection (Danswer), architectural overview, and migration plan.

Development will focus on integrating Danswer, setting up Telegram and Discord frontends, and migrating existing `faqqer` functionalities (blockchain stats, customer service analysis).
